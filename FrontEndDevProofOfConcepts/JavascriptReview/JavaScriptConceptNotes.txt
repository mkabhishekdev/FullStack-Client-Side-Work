6. "this" keyword in javascript
    
- this is a keyword that refers to the object that is currently calling the function.
      It helps functions access the properties of the object they belong to.
      It avoids repeating object names.
      It makes code reusable.
      Think of this as:
      👉 “Which object am I inside right now?”

      ✅ Simple Example Without this
let person = {
  name: "Abhi",
  age: 33,
  sayHello: function() {
    console.log("Hello, my name is " + person.name);
  }
};

person.sayHello(); // Hello, my name is Abhi

❌ Problem:
Inside sayHello, we used person.name.
If the object name changes, or if we reuse the function for another object, it breaks.

✅ Same example WITH this
let person = {
  name: "Abhi",
  age: 33,
  sayHello: function() {
    console.log("Hello, my name is " + this.name);
  }
};

person.sayHello(); // Hello, my name is Abhi

✔️ Why this is better:
this refers to the object person
Even if we rename the object, the function still works

🔄 Reusability Example

You can reuse the same function for multiple objects:

function introduce() {
  console.log("I am " + this.name);
}

let person1 = { name: "Abhi", introduce: introduce };
let person2 = { name: "Kiran", introduce: introduce };

person1.introduce(); // I am Abhi
person2.introduce(); // I am Kiran

✔️ How this works here:

When person1 calls the function → this becomes person1
When person2 calls the function → this becomes person2
